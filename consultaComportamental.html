O que é a Consulta Comportamental?

Geralmente são feitas em casos mais tranquilos ou para clientes que moram fora das regiões de atendimento.

Dura em torno de uma hora e meia cada visita.

O Objetivo é que o cliente fique capacitado para seguir um passo a passo para que sozinho resolva o problema comportamental do seu cão.

O cliente fica com minha disponibilidade via Whatsapp para mandar dúvidas e vídeos do cão para que a orientação sejao o mais personalizada possível.
